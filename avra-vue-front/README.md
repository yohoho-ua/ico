# avra-vue-front

> front for avra-rest-server

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:808x
npm run dev
```


``` bash
#OTHER:
# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run all tests
npm test
```
